*To use this app:*
- clone this repo  
- `cd First_api_server` 
- to download all dependencies used in this app, use in terminal 
> `npm i` 
- to run the server use in terminal  
> `nodemon` 
> try to use `api/v1/food` and `api/v1/clothes` to show data
- to run test it use in termintl 
> `npm test`
